# Joshua Samuel - Portfolio Website

Welcome to my personal portfolio website! This site showcases my skills, projects, and experience as a web developer and software engineer.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Interactive Navigation**: Smooth scrolling and mobile-friendly hamburger menu
- **Sections**:
  - Hero section with introduction
  - About me section
  - Skills showcase with modern tech stack
  - Projects portfolio
  - Contact information

## 🚀 Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome Icons

## 📦 Deployment

This website is hosted on GitHub Pages. To view the live site, visit:
[https://josh-winoflex.github.io/joshuasamuel/](https://josh-winoflex.github.io/joshuasamuel/)

## 🛠️ Local Development

To run this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Josh-WinOFlex/joshuasamuel.git
   cd joshuasamuel
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   python3 -m http.server 8080
   ```
   Then visit `http://localhost:8080`

## 📝 Customization

To customize this portfolio for your own use:

1. Update the content in `index.html` with your information
2. Modify colors and styles in `style.css`
3. Add your own projects and skills
4. Update contact information and social media links
